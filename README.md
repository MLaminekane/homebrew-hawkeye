# Homebrew Tap for Hawkeye

The flight recorder for AI agents - observability and security for Claude Code, Aider, AutoGPT and more.

## Install

```bash
brew tap MLaminekane/hawkeye
brew install hawkeye-ai
```

## Upgrade

```bash
brew upgrade hawkeye-ai
```
